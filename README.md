# @simonjang/create-with-template

NPM starter command to kick-start your node project.

> NOTICE: This release is still in a beta phase.

## Usage

When executing the command, you'll be asked to either select one of the templates that are provided for you, or to provide a custom template from a git repository.

```bash
npx @simonjang/create-with-template
```

### Options

| Flag | Description | Default |
|------|-------------|---------|
| `-g, --guided` | Use guided setup (interactive template selection) | `true` |
| `-v, --verbose` | Enable verbose logging | `false` |
| `-h, --help` | Show help | |

## Requirements

- Node.js >= 22
- Git >= 2.22
